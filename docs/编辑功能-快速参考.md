# 编辑功能快速参考卡片

> 一页纸快速上手指南

---

## 🎯 核心功能

### 1️⃣ 编辑角色描述

**在哪里**：剧本清单 → 左侧边栏 → 演员表  
**怎么做**：悬停角色卡片 → 点击 ✏️ → 修改 → 保存  
**影响**：素材准备阶段的角色参考图

### 2️⃣ 编辑画面提示词

**在哪里**：剧本清单 → 分镜列表 → 右侧提示词栏（或下方）  
**怎么做**：悬停提示词区域 → 点击 ✏️ → 修改 → 保存  
**影响**：导演工作台的关键帧图像

---

## ⚡ 快捷操作

| 操作 | 快捷键  | 说明         |
| ---- | ------- | ------------ |
| 编辑 | 点击 ✏️ | 进入编辑模式 |
| 保存 | 点击 ✓  | 确认并退出   |
| 取消 | 点击 ✗  | 放弃并退出   |
| 切换 | Tab     | 在元素间切换 |

## 📝 提示词模板

### 角色描述模板

```
[年龄/种族/性别] + [发型/外貌] + [服装] + [表情/姿态] + [风格关键词]

示例：
A 25-year-old Asian male with short black hair, wearing a
futuristic silver jacket, confident expression, athletic build,
photorealistic, cinematic lighting
```

### 画面提示词模板

```
[镜头类型] + [主体动作] + [环境] + [光线] + [镜头技术] + [风格]

示例：
Medium shot of character walking confidently through neon-lit
alley, rain-soaked pavement, blue and pink lighting, shallow
depth of field, anamorphic lens, cyberpunk aesthetic
```

## 🎨 常用关键词

### 镜头类型

`extreme close-up` `close-up` `medium shot` `wide shot` `establishing shot`

### 光线

`golden hour` `blue hour` `neon lighting` `soft diffused light` `dramatic shadows`

`photorealistic` `cinematic` `8K resolution` `highly detailed` `sharp focus`

## ✅ 检查清单

编辑完成后，检查以下项目：

- [ ] 所有角色描述是否足够具体？
- [ ] 关键分镜是否包含光线和氛围？
- [ ] 动作场景是否描述了镜头运动？
- [ ] 风格关键词是否与项目一致？
- [ ] 相邻分镜的描述是否连贯？

## 🚨 常见错误

### ❌ 太模糊

```
A person walking
```

### ✅ 具体明确

```
A 30-year-old woman with long brown hair walking confidently
down a busy city street at sunset, wearing a red coat
```

---

### ❌ 风格冲突

```
photorealistic, cartoon style, anime
```

### ✅ 风格统一

```
photorealistic, cinematic, film grain, IMAX quality
```

---

### ❌ 缺少细节

```
Interior scene, character talking
```

### ✅ 丰富细节

```
Interior of modern office, soft window light, character sitting
at desk talking on phone, shallow depth of field, professional
photography
```

## 💡 专业技巧

### 1. 角色一致性

在所有分镜中使用相同的角色描述关键词：

- 固定特征：`short black hair`, `athletic build`
- 可变元素：服装、表情、姿态

### 2. 场景连贯性

相邻分镜保持光线和氛围的合理过渡：

- 同一场景：相同的光线描述
- 时间推移：逐渐变化的光线（如 `golden hour` → `dusk` → `night`）

### 3. 关键词密度

重要的视觉元素重复提及 2-3 次：

```
neon-lit cyberpunk city, holographic displays, neon signs
reflecting on wet streets
```

### 4. 风格强化

每个提示词都包含项目风格关键词：

- 真人影视：`photorealistic, cinematic, 8K`
- 动漫：`anime style, cel-shaded, vibrant colors`
- 3D动画：`Pixar style, 3D CGI, stylized`
